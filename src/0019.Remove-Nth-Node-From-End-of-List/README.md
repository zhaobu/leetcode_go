# [删除链表的倒数第 N 个节点](https://leetcode-cn.com/problems/remove-nth-node-from-end-of-list/)

## 问题

### 描述

> 给定一个链表，删除链表的倒数第 _n_ 个节点，并且返回链表的头结点。

**示例：**

> ```text
> 给定一个链表: 1->2->3->4->5, 和 n = 2.
>
> 当删除了倒数第二个节点后，链表变为 1->2->3->5.
> ```

**说明：**

给定的 _n_ 保证是有效的

**进阶：**

你能尝试使用一趟扫描实现吗？
